# Tiny Rogues 简体中文汉化 v1.5.18

本版重点修复地面物品详情框空间不足，以及实机游玩中发现的动态文本漏翻。

## 更新内容

- 地面未拾取/待购买物品详情使用 82% 字号，宽度增加 20%，高度增加 100%
- 调整仅作用于带“拾取/购买”提示的地面物品，身上装备界面不受影响
- 补齐“士气高涨”“迎难而上”、士气状态、豪赌客和黑曜石骰子等文本
- 补齐彩虹护符、元素伤害、灯泡、骆驼缰绳、电鳗、骆驼等物品与同伴文本
- 修复逐字彩色富文本及分段教学组件无法完整匹配的问题

## 安装

1. 完全退出游戏。
2. 下载并解压发布压缩包。
3. 将压缩包内全部文件覆盖到《Tiny Rogues》游戏根目录。
4. 启动游戏；首次加载 BepInEx 可能需要稍候。
