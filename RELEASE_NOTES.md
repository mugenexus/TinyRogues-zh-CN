# Tiny Rogues 简体中文汉化 v1.5.25

本版集中补齐元进度节点、技能、装备和动态状态中的实机漏翻，并修复地面掉落详情显示问题。

## 更新内容

- 补齐全部 108 个元进度节点，以及风之步、旋刃、银河套装、灵魂商店等实机新增文本
- 修复文字方框、房间属性提示错位和地面掉落详情中英混排；身上装备界面保持原样
- 支持彩色逐字文本、局部翻译后的富文本和嵌套字号标签继续命中完整中文
- 当前 490 条实机样本未覆盖和英文残留均为 0

## 安装

1. 完全退出游戏。
2. 下载并解压发布压缩包。
3. 将压缩包内全部文件覆盖到《Tiny Rogues》游戏根目录。
4. 启动游戏；首次加载 BepInEx 可能需要稍候。
